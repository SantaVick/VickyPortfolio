My Story

Growing up, I was always curious about how things worked. Not just on the surface — but underneath. When I started programming, I quickly realized I was drawn to backend systems. While many people focus on what users see, I became interested in what users don’t see — the logic, the data flow, the security, and the architecture that keeps everything running.

I chose Java and Spring Boot because I wanted to work with technologies used in serious, production environments. I didn’t just want to build apps — I wanted to build systems that businesses can rely on.

I believe technology should work so well that you don't even notice it. It should just... work. Every time.
